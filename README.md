# Sistema de Gestión de Pedidos

## Descripción
Este proyecto es un sistema web desarrollado para **gestionar pedidos** de manera eficiente. Permite registrar, visualizar, actualizar y eliminar pedidos, así como llevar un control de clientes, repartidores, montos y horarios de entrega. Está diseñado para mejorar la organización y la productividad de negocios que manejan pedidos frecuentes.

---

## Funcionalidades

- Registro y gestión de pedidos diarios.  
- Control de clientes y repartidores.  
- Registro de hora de salida y llegada de pedidos.  
- Cálculo de montos totales y resúmenes semanales/mensuales.   
- Diseño responsivo para dispositivos móviles y escritorio.  
- Posibilidad de generar reportes de pedidos.

---

## Tecnologías utilizadas

- **Frontend:** HTML, CSS, Bootstrap, JavaScript  
- **Base de datos:** MySQL  
- **Control de versiones:** Git y GitHub
